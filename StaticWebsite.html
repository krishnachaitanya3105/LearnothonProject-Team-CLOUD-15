Static Website Optimization

Using Google Cloud CDN & Caching

Delivering fast, scalable, and globally optimized web experiences using Google Cloud Platform.

📌 Project Overview

This project demonstrates how a static website hosted on Google Cloud Storage can be optimized for global performance using Cloud CDN. Content is cached at edge locations worldwide, enabling faster load times and reduced latency for users regardless of geographic location.

🏗️ System Architecture

⚠️ Problem

Static websites in a single region suffer high latency for distant users.

✅ Solution

Cloud CDN caches content globally for fast delivery and scalability.

🛠️ Implementation Steps

  1. Create a Cloud Storage bucket
  2. Upload static website files
  3. Configure bucket for public website hosting
  4. Create External HTTP Load Balancer
  5. Add backend bucket
  6. Enable Cloud CDN
  7. Test global performance

🏁 Conclusion

Google Cloud CDN provides a scalable solution for optimizing static websites by caching content at global edge locations.
